Warehouse Inventory & Order Management System <br>
Overview <br>
This project is a lightweight Python-based warehouse inventory and order management system designed to demonstrate practical use of core data structures. It simulates real-world warehouse operations such as inventory tracking, order processing, low-stock monitoring, and storage optimization.
The system emphasizes efficiency, input validation, and clear handling of edge cases, making it a strong example of applying data structures to solve business problems. <br>
Features <br>
-Inventory lookup by SKU <br>
-Inventory updates with validation <br>
-FIFO order processing using a queue <br>
-Low stock detection with sorted results <br>
-Intelligent warehouse zone recommendations <br>
Technologies Used <br>
-Python 3.11 <br>
-Built-in data structures: <br>
--Dictionaries (hash tables) <br>
--Queues (collections.deque) <br>
--Lists (sorting and filtering) <br>
Data Structures & Design <br>
This project demonstrates how different data structures can be used effectively in a logistics context: <br>
-Hash Tables (Dictionaries) <br>
--Used for fast SKU lookup and inventory updates. <br>
-Queue (Deque) <br>
--Implements first-in-first-out (FIFO) order processing. <br>
-Sorted Lists <br>
--Used to organize low-stock items by urgency. <br>
-Nested Dictionaries <br>
--Represent warehouse zones with capacity and usage tracking. <br>
